---
title: Installing Teleport
description: Provides information on installing Teleport and keeping your Teleport installations up to date.
---

The guides in this section show you how to manage a Teleport installation.

- [Install Teleport](./install/install-teleport.mdx): Guides to installing
  Teleport binaries on various systems.
- [Run Teleport as a Daemon](./install/daemon.mdx): Configure a Teleport
  installation to run as a systemd service. Note that installing Teleport as a
  package already sets up a systemd service on most Linux distributions.
- [Uninstall Teleport](./install/uninstall-teleport.mdx): Guides to removing
  Teleport installations from a system.
